## Install
NPM

```bash copy
npm install -g myplop
```

PNPM

```bash copy
pnpm add myplop
```

Yarn

```bash copy
yarn add myplop
```

## Install App (Required)
Choose `App` option
NPM

```bash copy
npm myplop init 
```

PNPM

```bash copy
pnpm myplop init 
```

Yarn

```bash copy
yarn myplop init 
```
